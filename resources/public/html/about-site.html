About this site

An Ecosystem of Trust (EoT) reference example of a participant's site which is part of an Information Sharing Network (ISN).

What it does

This site is part of an EoT Border Trade Demonstrator (BTD) ISN. It represents a participants site which can be used to share signals into the ISN.

Participant's sites can be used to publish signals into an ISN. Others can login and view signals on your site when access control permits it.

For more information on how the ISN works please see the documentation.
