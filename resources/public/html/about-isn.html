About this ISN network site

An Ecosystem of Trust (EoT) reference example of a network site which is part of an Information Sharing Network (ISN).

What it does

This network site is part of an EoT Border Trade Demonstrator (BTD) ISN. It is the entry point for the ISN.

It provides information on:

  • the purpose of the ISN and the categories of information being exchanged (TODO: TBD)
  • a number of participants working within the ISN contributing and consuming mutually beneficial information
  • a number of mirror sites which let information be sydicated to aggregate intelligence for convenient surfacing
  • TODO: TBD a form of governance framework (Collaboration Agreement) which has been calibrated to govern transactions within the ISN on several dimensions including; legal basis to share, liability etc

The network site may be used to add participant sites and mirrors to the ISN via API.
