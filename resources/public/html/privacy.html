Privacy

This Information Sharing Network site stores signal data only.

Any signals created using the API provided by this site are stored on the filesystem of the machine the site is running on (by default).

Personally Identifiable Information (PII) would only ever be accessed or stored if an accompanying Collaboration Agreement exists, is associated with this site and is setup to provision for access and storage of PII.

The Indieauth compatible service used for authentication and authorisation for early ISN sites processes and stores the following information:

  • the profile URL ("me") entered
  • the URL, name and redirect URL for the application being granted access
  • the name of any authentication providers found and verified from "me" (twitter, github etc)
  • the associated profile URL for those providers, or the URL to a PGP key
  • the date of authentication
